Countdown.js

Create a countdown character counter for a field.

Example

Code


<form id="form">
	<textarea name="text" id="text" maxlength="20"></textarea>
</form>

$('textarea').countdown({thresholds: {low: 0.2, over: 0}});

Dependencies

  • jQuery
  • jQuery Widget Factory
  • String.pluralize

Config Object

max
Maximum number of characters allowed in field. Default: field maxlength value
'class'
Class to apply to the countdown counter. Default: counter
text
The text to put after the counter. Assumes first word should be plural or singular based on count left. Default: 'character left'.
thresholds
Object of thresholds of counter number (as a percentage of the max in decimal format) to apply classes to the counter. Default: {}
events
Array of events to trigger the recalculating of the counter. Events are namespaced with countdown. Also adds a custom event of reCount. Default: ['keyup', 'blur', 'focus', 'change', 'paste']
callback
Custom function to call after the recalculation of the count. Default: $.noop

Public Methods

build
Build out the counter HTML and attach it the DOM.
getCount
Get the characters left from the max minus the number of characters in the input.
getThreshold
Get what threshold (if any) the value has surpassed.
