String.pluralize

Pluralize a singular term based on a count provided and optionally override the generated plural version with a provided plural form.

'test'.pluralize(1) // return '' 'test'.pluralize(2) // return '' 'person'.pluralize(1, 'people') // return '' 'people'.pluralize(2, 'people') // return '' 'fax'.pluralize(2) // return '' 'mess'.pluralize(2) // return '' 'pretty'.pluralize(2) // return ''
