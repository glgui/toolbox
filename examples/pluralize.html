Pluralize

Pluralize a singular term based on a count provided and optionally override the generated plural version with a provided plural form.

'test'.pluralize(1) // return 'test' 'test'.pluralize(2) // return 'tests' 'person'.pluralize(1, 'people') // return 'person' 'people'.pluralize(2, 'people') // return 'people' 'fax'.pluralize(2) // return 'faxes' 'mess'.pluralize(2) // return 'messes' 'pretty'.pluralize(2) // return 'pretties'
